## Installation

1. Create the database tables
```bash
npm run db:create
# or
yarn db:create
```

2. Run the migration command
```bash
npm run db:migrate
# or
yarn db:migrate
```

3. Run the seed command
```bash
npm run db:seed
# or
yarn db:seed
```

**Ignore if using docker**\
4. Run the server
```bash
npm run dev
# or
yarn dev
```
